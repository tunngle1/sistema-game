/*
  TILDA ZERO BLOCK — ИНСТРУКЦИЯ ПО ЗАГРУЗКЕ
  ==========================================

  ФАЙЛЫ ПРОЕКТА
  -------------
  index.html      — главная страница
  privacy.html    — политика конфиденциальности (отдельная страница в Tilda)
  styles.css      — все стили
  script.js       — логика (модалки, отзывы, оплата)
  config.js       — НАСТРОЙКИ: цена, ссылка на оплату, видеоотзывы

  СПОСОБ 1 — Zero Block (полная копия дизайна)
  ---------------------------------------------
  1. Создайте страницу в Tilda
  2. Добавьте блок T123 «Zero Block»
  3. HTML — содержимое index.html (без <head>, только body)
  4. CSS  — styles.css
  5. JS   — config.js + script.js (в таком порядке!)
  6. Шрифты: Cormorant Garamond, Manrope

  СПОСОБ 2 — Стандартные блоки Tilda (рекомендуется для оплаты)
  ---------------------------------------------------------------
  Соберите страницу из блоков Tilda, используя index.html как референс:

  Секция          | Блок Tilda
  ----------------|------------------------------------------
  Hero            | CR16 / CR01 (обложка)
  О игре          | TX12 (текст + изображение)
  4 уровня        | FR204 (карточки) или GL03
  Для кого        | TX16 (список)
  Что получите    | FR204 (иконки)
  Видеоотзывы     | VD01 (видео) × 3  или  GL18 (галерея)
  Автор           | TX12 + фото
  Оплата          | ST100 (Store) или PR100 (Pricing) + Tilda Pay
  Форма           | BF204N (форма) + подключить Tilda Payments
  Политика        | отдельная страница TX01, текст из privacy.html

  ОПЛАТА — 3 ВАРИАНТА
  --------------------

  A) Tilda Payments (самый простой на Tilda):
     1. Настройки сайта → Платежи → подключить ЮKassa / Stripe
     2. Блок ST100 «Товары» или PR100 «Тарифы» — создать товар «Участие в игре»
     3. Форма BF204N → вкладка «Платежи» → привязать товар
     4. В config.js на Zero Block-версии вставьте paymentUrl из Tilda

  B) ЮKassa (прямая ссылка):
     1. Создайте платёжную ссылку в личном кабинете ЮKassa
     2. Вставьте в config.js → payment.paymentUrl

  C) Prodamus / GetCourse / Тинькофф:
     1. Создайте продукт и получите ссылку на оплату
     2. Вставьте в config.js → payment.paymentUrl

  ВИДЕООТЗЫВЫ
  -----------
  Вариант A (Zero Block): отредактируйте config.js → reviews[]
    - videoUrl: embed-ссылка YouTube / Rutube / VK
    - name, role, quote — текст под видео

  Вариант B (Tilda): блок VD01 «Видео» × 3
    - Загрузите видео на YouTube/VK и вставьте ссылки
    - Или блок GL18 с превью и ссылками

  ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ
  ----------------------------
  1. Создайте вторую страницу в Tilda (например /privacy)
  2. Скопируйте текст из privacy.html
  3. В футере главной страницы — ссылка на /privacy
  4. В форме регистрации — галочка согласия со ссылкой

  НАСТРОЙКА config.js
  -------------------
  payment.price       — цена (15000)
  payment.paymentUrl  — ссылка на оплату (ОБЯЗАТЕЛЬНО перед запуском!)
  payment.successUrl  — страница «Спасибо» после оплаты
  reviews[]           — массив видеоотзывов
  links.telegram      — ссылка на Telegram
  links.privacy       — ссылка на политику (/privacy)

  ЦВЕТА
